# Liborrow

Liborrow is a simple and efficient library borrowing management system designed to streamline the process of tracking books, borrowers, and lending activities. It was built to simplify library operations with a clean structure and easy-to-use functionality.

---

## 📚 Overview

Liborrow helps manage:

- 📖 Books and inventory
- 👤 Borrower records
- 🔄 Borrowing and return transactions
- ⏰ Due dates and tracking

The goal of the project was to create a lightweight and practical solution for small libraries, student projects, or learning purposes.

---

## 🚀 Features

- Add, update, and remove books
- Register and manage borrowers
- Borrow and return books
- Track due dates
- Prevent borrowing of unavailable books
- Maintain borrowing history
- Simple and intuitive structure
